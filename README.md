# multicriteria_methods

Multiple-criteria decision-making using AHP, ELECTRE ..  

### Installing

Activate virtual env


```
pipenv shell
```

Install prerequisites

```
pipenv install
```

* [Pipenv](https://pipenv.readthedocs.io/en/latest/) - Dependency Management
